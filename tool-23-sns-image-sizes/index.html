SNS画像サイズ一覧について

SNSに画像を投稿する際、推奨サイズに合わせることで画質の劣化やトリミングを防ぎ、最適な表示が可能になります。各プラットフォームによって推奨サイズが異なるため、投稿前に確認しておくことが重要です。

X（旧Twitter）の画像サイズ

X（Twitter）の投稿画像は1200×675px（16:9）が推奨です。2枚投稿の場合は700×800px（7:8）になります。ヘッダー画像は1500×500px（3:1）、プロフィール画像は400×400px（1:1）です。画像が推奨サイズと異なると自動トリミングされるため、重要な情報が切れてしまうことがあります。

Instagramの画像サイズ

Instagramのフィード投稿は正方形1080×1080px（1:1）が基本ですが、縦長1080×1350px（4:5）の方がフィードで大きく表示されるため、エンゲージメントが高くなる傾向があります。ストーリーズとリールは1080×1920px（9:16）の縦型フルスクリーンです。

YouTubeの画像サイズ

YouTubeサムネイルは1280×720px（16:9）が推奨です。サムネイルは動画のクリック率に直結するため、推奨サイズで高画質な画像を作成することが重要です。チャンネルアートは2560×1440pxですが、デバイスによって表示範囲が変わるため、重要な情報は中央の1546×423pxに収めましょう。

LINE・Facebook・OGPのサイズ

LINEリッチメニュー（大）は2500×1686px、リッチメッセージは1040×1040pxです。Facebookの投稿画像は1200×630px、カバー画像は851×315pxが推奨です。OGP（Open Graph Protocol）画像は1200×630pxで、SNSでURLがシェアされた時に表示されるプレビュー画像に使われます。

画像作成のコツ

推奨サイズで作成した画像はファイルサイズが大きくなりがちです。投稿前に画像を圧縮することで、アップロード時間の短縮と表示速度の向上が期待できます。サイズカードをクリックすると、サイズ情報をクリップボードにコピーできます。

noteの見出し画像（アイキャッチ）サイズ

note（ノート）の記事トップに表示される見出し画像（アイキャッチ）の推奨サイズは1280×670px（約1.91:1）です。これはOGP画像やX（Twitter）でシェアされた際のカード比率とほぼ同じで、1枚作っておけばSNSシェア時のサムネイルとしても流用できます。スマホ・PCで見切れにくく、記事一覧でもきれいに表示されます。文字を入れる場合は、上下が少しトリミングされても読めるよう中央寄せにするのがおすすめです。

よくある質問

X（旧Twitter）の投稿画像の推奨サイズは？
X（旧Twitter）の投稿画像は1200×675px（16:9）が推奨です。ヘッダー画像は1500×500px（3:1）、プロフィール画像は400×400px（1:1）が目安です。推奨サイズと異なると自動トリミングされる場合があります。
Instagramのフィード投稿はどのサイズが最適ですか？
正方形1080×1080px（1:1）が基本ですが、縦長1080×1350px（4:5）の方がフィードで大きく表示されます。ストーリーズやリールは1080×1920px（9:16）の縦型フルスクリーンが推奨です。
noteの見出し画像の推奨サイズは？
noteの見出し画像（アイキャッチ）は1280×670px（約1.91:1）が推奨です。OGPやSNSシェア時のカード比率とほぼ同じため、1枚でSNSのサムネイルにも流用できます。
YouTubeのサムネイルのサイズは？
YouTubeのサムネイルは1280×720px（16:9）が推奨です。クリック率に直結するため、推奨サイズで高画質に作成し、小さく表示されても読める大きめの文字を使うのが効果的です。
OGP画像の推奨サイズは？
OGP（Open Graph Protocol）画像は1200×630px（約1.9:1）が推奨です。URLがSNSでシェアされた際に表示されるプレビュー画像で、この比率にしておくとX・Facebook・LINEなどで見切れにくくなります。
