ハッシュ生成ツールについて

テキストから各種ハッシュ値を生成するツールです。MD5、SHA-1、SHA-256、SHA-512に対応。パスワードのハッシュ確認、ファイルの整合性チェック、デジタル署名の検証などに利用できます。

Web Crypto APIを使用しており、すべての処理はブラウザ内で完結します。SHA系のハッシュ生成にはブラウザのネイティブ暗号化機能を使用するため、高速かつ安全です。
