HTMLカラーネーム一覧

140色のHTMLカラーネームをビジュアル表示

HTMLカラーネーム一覧の使い方

HTML/CSSで使用できる140色以上の名前付きカラー（Named Colors）を一覧表示する無料ツールです。色名・HEXコード・RGB値をまとめて確認でき、クリックでカラーコードをコピーできます。

名前付きカラーとは

CSSでは「red」「blue」「skyblue」「coral」などの英語の色名でカラーを指定できます。HEXコードを覚えなくても直感的に色を指定できるため、プロトタイプの作成やデバッグ時に便利です。

検索・フィルター機能

色名やHEXコードで検索できるほか、赤系・青系・緑系などの色相でフィルタリングもできます。デザインの色選びの参考にも使えます。

基本の16色（HTML標準カラー）早見表

もっとも古くから使われている16色の基本カラーネームです。色名・色見本・HEXコードをまとめました。HEX値はCSSの標準仕様で定義されており、主要ブラウザで同じ色に表示されます。

よくある質問

CSSで使える色名はいくつありますか？
現在のCSS仕様では140色前後の名前付きカラーが定義されています（CSS Color Module Level 4では148色）。本ツールではその一覧を色見本付きで確認でき、クリックでHEXコードをコピーできます。
HTMLカラーネームはどのブラウザでも同じ色で表示されますか？
はい。redやskyblueなどの名前付きカラーはCSSの標準仕様で各色のHEX値が定義されているため、主要なモダンブラウザでは同じ色で表示されます。厳密に色を管理したい場合はHEXコードの併用がおすすめです。
grayとgrey、どちらの綴りでも使えますか？
どちらも有効です。CSSではアメリカ式のgrayとイギリス式のgreyの両方の綴りがサポートされており、DarkGray／DarkGreyなども同様に両方使えます。表示される色は同じです。
色名とHEXコードはどちらを使うべきですか？
可読性を重視するプロトタイプやサンプルでは色名が便利です。一方、デザインで微妙な色合いを正確に指定したい場合や、デザインシステムで色を統一したい場合はHEXコード（またはRGB）の利用が向いています。
redとcrimsonの違いは何ですか？
redは純粋な赤（#FF0000）、crimsonはやや深く青みがかった赤（#DC143C）です。同じ「赤」でもHEX値が異なるため、デザインの雰囲気に合わせて使い分けられます。本ツールで色見本を見比べて選べます。
